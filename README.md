learn golang and algorithms with leetcode

> 
> **目标**：刷题的过程中，结合《算法 4th》的内容，补充上自己算法与数据结构知识的盲点，以及知识点与真实编码之间的 gap，并熟练掌握
>
> *先进行尝试，不要拘泥于格式*，开始的时候，最重要的是要坚持
>
> 先坚持 100 天试试
